# Open Graph Dynamic SVG Image

## Overview

The Parth Nation website now uses a dynamically generated SVG image for social media sharing (Open Graph and Twitter Cards). This means the preview image is generated on-the-fly based on information rather than using a static image file.

## How It Works

The OG image is generated by an API route at `/api/og` that creates an SVG with the following features:

- **Dynamic gradient background** in the site's slate/navy color scheme
- **Basketball-themed decorations** with emoji and subtle patterns
- **Customizable text content** via URL parameters
- **Professional layout** optimized for social media (1200x630px)
- **Brand colors** matching the orange/amber theme

## Default Image

By default, when someone shares `https://parthnation.com`, they'll see an image with:

- **Title**: "Parth Nation"
- **Subtitle**: "Honoring the Heart of Charlton Basketball"
- **Date**: "December 13th, 2025"
- **Location**: "Shepherd Hill Regional High School, Dudley, MA"
- **Call to Action**: "Join the Tournament"
- **Tagline**: "Supporting Mental Health Awareness"

## Custom Parameters

You can customize the OG image by adding query parameters:

### Examples:

#### Custom title
```
/api/og?title=Special%20Event
```

#### Custom subtitle
```
/api/og?subtitle=Community%20Celebration
```

#### Custom date
```
/api/og?date=January%201st,%202026
```

#### Custom location
```
/api/og?location=Boston,%20MA
```

#### All custom parameters
```
/api/og?title=Parth%20Nation%202026&subtitle=Annual%20Tournament&date=June%2015th,%202026&location=Worcester,%20MA
```

## Testing the Image

You can view the generated SVG directly in your browser:

1. **Local development**: `http://localhost:3000/api/og`
2. **Production**: `https://parthnation.com/api/og`

## Technical Details

- **Format**: SVG (Scalable Vector Graphics)
- **Dimensions**: 1200x630 pixels (optimal for social media)
- **Runtime**: Edge runtime for fast generation
- **Caching**: Immutable cache for performance (31536000 seconds = 1 year)
- **Content-Type**: `image/svg+xml`

## Social Media Compatibility

The SVG format is supported by:
- ✅ Facebook
- ✅ Twitter/X
- ✅ LinkedIn
- ✅ Discord
- ✅ Slack
- ✅ Most modern social platforms

## Benefits

1. **Always up-to-date**: Change content without regenerating image files
2. **Small file size**: SVG is much smaller than PNG/JPG
3. **Crisp on all displays**: Vector format scales perfectly
4. **Easy to maintain**: Update code instead of editing images in design software
5. **Consistent branding**: Automatically matches site theme
